# **App Name**: CodeFolio

## Core Features:

- Project Showcase: A dynamic section to display detailed information about software engineering projects, including descriptions, technologies used, and links to live demos or repositories.
- About Me Section: A dedicated page to present the engineer's professional biography, skills, work experience, educational background, and integrated contact/social media links.
- Contact Form & Details Display: A functional form allowing visitors to send inquiries or messages directly to the portfolio owner, alongside a clear display of key contact information (e.g., email, LinkedIn).
- AI-Powered Project Description Tool: An integrated AI tool to assist the engineer in generating compelling and concise project descriptions by highlighting key achievements and technologies.
- Profile Sharing: Allows users to easily share a link to the portfolio profile across various platforms.
- External Links Integration: Seamless integration and display of links to professional profiles and resources such as LinkedIn, GitHub, or personal blogs.

## Style Guidelines:

- Primary color: A modern and professional vibrant blue-green (#33B2EB), signifying innovation and clarity. The color's brightness creates a good contrast against a dark background.
- Background color: A deep, almost black, blue-gray (#161A1B) with a subtle hint of blue, providing a sleek and focused aesthetic suitable for showcasing technical content.
- Accent color: A vibrant purple (#8A2BE2) for interactive elements and calls to action, providing a striking contrast against the dark background, reflecting the button colors in the image.
- Hero section background: A dark, subtle reddish gradient transitioning from a deeper red to black, creating a dramatic backdrop for the profile image.
- Headlines font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and slightly angular feel, ensuring headlines are impactful.
- Body font: 'Inter' (sans-serif) for its high readability and neutral, professional aesthetic, suitable for extensive project descriptions and bio content.
- Code font: 'Source Code Pro' (monospace sans-serif) for elegant and readable display of code snippets.
- Utilize modern, minimalist linear icons for navigation, social media links (including LinkedIn), and interactive elements, maintaining a clean and professional appearance.
- Adopt a clean, card-based layout for sections like projects and profile details, with ample whitespace to ensure content is easy to scan and comprehend without clutter.
- Ensure the entire site layout is fully responsive, adapting seamlessly and displaying optimally across various devices, from desktop to mobile.
- Implement subtle hover effects and smooth transitions for interactive elements, enhancing user experience without distracting from content.
